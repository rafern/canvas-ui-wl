Class: module:WLVirtualKeyboardRoot

module:WLVirtualKeyboardRoot(wlObject, material, keyboardTemplateopt, keyboardDriveropt, themeopt, unitsPerPixelopt, collisionGroupopt, registerPointerDriveropt)

A WLRoot with a virtual keyboard, similar to VirtualKeyboardRoot from canvas-ui. Can also be automatically hidden when there is no keyboard focus by callid updateVisibility before calling update. This doesn't need to be directly used, the virtual-keyboard-ui-root component can be used instead.

 1 The collision group that this root's collider will belong to. If null, collider and cursor-target will not be added.

 true Register the default pointer driver to this root? If collisionGroup is null, this is forced to false.
